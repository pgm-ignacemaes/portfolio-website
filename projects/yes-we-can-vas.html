Yes, we Can...vas!

Voor deze eindproject ontwikkelde ik een Single Page Application met React, Typescript en Tanstack voor het beheren van studenten en vakken binnen de graduaatsopleiding Programmeren aan de Arteveldehogeschool.

De applicatie laat gebruikers toe om vakken en studenten te beheren via een simpele interface. Hierbij wordt gebruik gemaakt van een mockAPI voor het opslaan en ophalen van data.

Je kan studenten en vakken toevoegen aan de API via formulieren. Deze zijn gemaakt met Tanstack form op basis van de datastructuur van een object. Indien er fouten zijn binnen het form wordt een gepaste error getoond.

De studenten kunnen verwijdert woden. Wanneer je op de delete knop duwt wordt een dialog getoond die vraagt of je deze student wilt verwijderen. Wanneer je op oke klikt wordt de pagina herladen en is de student verwijdert.
