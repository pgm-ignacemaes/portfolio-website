Mockify

Voor dit project ontwikkelde ik een REST API met behulp van Express. De focus lag op het bouwen van een gestructureerde en schaalbare backend waarin muziekdata beheerd kan worden via verschillende endpoints.

De API bevat twee resources: Tracks en Playlists. Beide moesten gegevens kunnen tonen, toevoegen, updaten en verwijderen. Bij het ophalen van lijsten kan de gebruiker een query parameter gebruiken om de resultaten oplopend of aflopend te sorteren.

De API controleert alle inkomende data en geeft correcte HTTP statuscodes terug. Wanneer een input niet voldoet aan de vereisten, wordt een duidelijke foutmelding teruggestuurd.
